# SQLHelper
新人学习数据库时制作的工具库

## 已实现

* 连接 MySQL 数据库
* 连接 SQLite 数据库
* `SELECT` 语句
* `UPDATE` 语句
* `INSERT` 语句
* `DELETE` 语句
* 条件 (`WHERE` 语句)

## 示例

[吾爱破解](https://www.52pojie.cn/thread-1629732-1-1.html)

## 在你的项目使用

![jitpack](https://jitpack.io/v/MrXiaoM/SQLHelper.svg)

Gradle:
```buildscript
repositories {
    maven { url 'https://jitpack.io' }
}
dependencies {
    implementation 'com.github.MrXiaoM:SQLHelper:1.4.1'
}
```
